# 다이렉트 보험 사이트

삼성생명 다이렉트 사이트를 참고하여 제작한 온라인 보험 소개 및 보험료 계산기 웹사이트입니다.

## 주요 기능

### 1. 보험상품 소개
- 실손의료보험
- 암보험
- 운전자보험
- 치아보험
- 종신보험
- 연금보험

각 상품마다 상세한 설명, 특징, 예상 보험료를 제공합니다.

### 2. 보험료 계산기
사용자 정보를 입력하여 예상 보험료를 실시간으로 계산할 수 있습니다.

**입력 항목:**
- 보험 종류
- 성별
- 나이
- 보장금액
- 보장기간
- 흡연 여부

**계산 로직:**
- 보험 종류별 기본 보험료
- 나이에 따른 차등 적용
- 성별에 따른 조정
- 보장금액 및 기간에 따른 계산
- 흡연자 할증 적용

### 3. 반응형 디자인
- 데스크톱, 태블릿, 모바일 등 다양한 화면 크기에 최적화
- 모바일 퍼스트 접근 방식

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**:
  - CSS Grid & Flexbox 레이아웃
  - CSS 변수를 활용한 테마 관리
  - 애니메이션 및 트랜지션
  - 반응형 미디어 쿼리
- **JavaScript (Vanilla)**:
  - DOM 조작
  - 이벤트 핸들링
  - 동적 콘텐츠 생성
  - Intersection Observer API

## 실행 방법

1. 프로젝트 폴더로 이동
```bash
cd insurance-direct-site
```

2. `index.html` 파일을 브라우저에서 열기
   - 파일 탐색기에서 `index.html` 더블클릭
   - 또는 브라우저 주소창에 파일 경로 입력
   - 또는 Live Server 등의 개발 서버 사용

## 주요 섹션 설명

### 헤더 (Header)
- 로고 및 브랜드명
- 네비게이션 메뉴
- 로그인/회원가입 버튼

### 히어로 섹션 (Hero)
- 메인 배너
- 주요 메시지 전달
- CTA 버튼

### 상품 섹션 (Products)
- 6개 보험상품 카드
- 각 상품별 특징 및 가격
- 가입하기 버튼

### 계산기 섹션 (Calculator)
- 입력 폼
- 실시간 보험료 계산
- 결과 표시

### 장점 섹션 (Benefits)
- 다이렉트 보험의 장점 4가지
- 아이콘 및 설명

### 푸터 (Footer)
- 회사 정보
- 고객센터 정보
- 바로가기 링크
